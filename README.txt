Cara menjalankan project Association Skin Care:

1. Buka Anaconda Navigator, Jupyter Notebook, JupyterLab, atau VS Code.
2. Buka file Association.ipynb.
3. Pastikan file skin_care.csv berada dalam folder yang sama dengan Association.ipynb.
4. Jalankan cell dari atas ke bawah atau pilih Run All.

Catatan:
- Project ini menggunakan dataset skin_care.csv.
- Notebook menggunakan pendekatan Association Rule Mining tanpa library mlxtend, jadi cukup menggunakan pandas, numpy, dan matplotlib.
- Output utama: frequent_itemsets.csv dan association_rules.csv.
